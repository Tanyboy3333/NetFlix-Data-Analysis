#  Movie Data Analysis (EDA Project)

## 📌 Overview

This project performs **Exploratory Data Analysis (EDA)** on a movie dataset to uncover insights about genres, popularity, and voting trends.
The goal is to clean the data, transform it, and extract meaningful patterns using visualization.

---

##  Key Questions Answered

* What is the most common genre?
* Which movies are the most and least popular?
* How are vote averages distributed?
* Which year had the highest number of movie releases?

---

##  Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Key Insights

* 🎭 Drama is the most frequent genre (~14%)
* 🕷️ *Spider-Man: No Way Home* has the highest popularity
* 📅 2020 had the highest number of movie releases
* ⭐ Vote averages are evenly distributed across categories


